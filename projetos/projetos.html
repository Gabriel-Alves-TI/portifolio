MEUS PROJETOS

Controle Financeiro

Aplicação web em C# com arquitetura MVC, Entity Framework e SQL Server. Conta com autenticação de usuários, recuperação de senha via e-mail (SMTP Gmail) e criptografia de senhas. Permite gerenciar lançamentos, categorias, contas e exibe dashboards de balanço. Módulos de Metas e Orçamento estão em desenvolvimento.

Projeto Serralheria

Este projeto foi desenvolvido com o objetivo de atender às necessidades reais de um negócio familiar do ramo de serralheria, permitindo o gerenciamento de orçamentos e recibos de pagamento. Esta versão representa um modelo simplificado, criado para fins de estudo, demonstração e entendimento do funcionamento do sistema real.

Todo List App

Este projeto foi feito com os conhecimentos adquiridos no curso de Javascript do Matheus Battisti - Hora de Codar. Fiz algumas adaptações no script e desenvolvi uma API em NodeJS utilizando o ORM Sequelize com MySQL.

Pokedex - v1

Desenvolvi esse projeto em Single Page Application para praticar Fetch API, fazendo requisições HTTP GET diretamente da "https://pokeapi.co"

Projeto E-commerce

Este é um projeto de e-commerce desenvolvido para gerenciar produtos e pedidos. Ele inclui funcionalidades básicas para operações CRUD, integração com banco de dados utilizando o ORM Sequelize.
